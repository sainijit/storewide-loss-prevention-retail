# Store-wide Loss Prevention

A suite of Intel® edge AI applications for retail loss prevention, powered by Intel® SceneScape spatial computing and Intel® OpenVINO™ inference.

## Applications

- **[Person of Interest (POI) Re-identification](./person-of-interest/README.md)**: Real-time detection of enrolled suspects across multiple cameras using face re-identification and FAISS vector search, with historical investigation capabilities.
- **[Suspicious Activity Detection](./suspicious-activity-detection/)**: Behavioral analysis for detecting suspicious activities in retail environments.
